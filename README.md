A real-time AI application that detects facial emotions using a webcam and overlays custom persona effects. Built with DeepFace, OpenCV, and Python, this project allows users to visualize emotions dynamically on their faces with fun filters.

Features

Real-time facial emotion detection:
Detects happy, sad, angry, surprise, neutral emotions.

Persona overlays:
Apply creative visual effects to your webcam feed.

Countdown capture:
Take photos with a timed sequence for perfect shots.
